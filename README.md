# CodeVedX Data Science Internship Projects

This repository contains the projects completed during my Data Science Internship at CodeVedX. These projects demonstrate the application of Machine Learning, Data Analysis, and Predictive Modeling using Python.

## Projects

### 1. Iris Flower Classification
- Built a machine learning model to classify Iris flowers into different species.
- Algorithm: K-Nearest Neighbors (KNN)
- Dataset: Iris Dataset

### 2. Titanic Survival Prediction
- Developed a predictive model to determine passenger survival on the Titanic.
- Algorithm: Logistic Regression
- Dataset: Titanic Dataset

### 3. Sales Prediction Using Python
- Created a regression model to predict sales based on advertising expenditure.
- Algorithm: Linear Regression
- Dataset: Advertising Dataset

### 4. Movie Rating Prediction
- Built a machine learning model to predict movie ratings using various movie attributes.
- Techniques: Data Preprocessing, Feature Engineering, Regression Analysis
- Dataset: IMDb Movies India Dataset

### 5. Credit Card Fraud Detection
- Developed a classification model to identify fraudulent credit card transactions.
- Techniques: Data Cleaning, Feature Scaling, Fraud Detection
- Dataset: Credit Card Fraud Detection Dataset
- Note: Dataset file is not included due to GitHub file size limitations.
