# CodeVedX Data Science Internship Projects

This repository contains the projects completed during my Data Science Internship at CodeVedX.

## Projects Included

### 1. Iris Flower Classification
- Built a machine learning model to classify Iris flowers into different species.
- Algorithms Used: K-Nearest Neighbors (KNN)
- Tools: Python, Pandas, NumPy, Scikit-learn, Matplotlib

### 2. Titanic Survival Prediction
- Developed a model to predict passenger survival on the Titanic dataset.
- Algorithms Used: Logistic Regression
- Tools: Python, Pandas, NumPy, Scikit-learn, Seaborn

### 3. Sales Prediction Using Python
- Created a regression model to predict sales based on advertising data.
- Algorithms Used: Linear Regression
- Tools: Python, Pandas, NumPy, Scikit-learn, Matplotlib
